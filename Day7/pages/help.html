用户管理系统使用指南

1. 系统简介

用户管理系统是一个基于 Flask 框架开发的 Web 应用程序，提供用户注册、登录、个人信息管理、文件上传、用户搜索、留言板、密码修改等功能。本系统主要用于网络安全实训教学，帮助理解常见的 Web 安全漏洞及其防护方法。

2. 账号注册与登录

新用户可以通过注册页面创建账号。注册时需要填写用户名、密码、邮箱和手机号码。已注册用户可使用用户名和密码登录系统。

默认测试账号：

  • 管理员：admin / admin123
  • 普通用户：alice / alice2025

3. 个人信息查看与修改密码

登录后可在首页查看个人基本信息。点击"个人中心"可查看更详细的资料，并在个人中心页面修改登录密码。

4. 用户搜索

在首页的搜索框中输入关键词（用户名或邮箱），点击搜索按钮即可查找系统中的用户。支持模糊匹配。

5. 文件上传

登录后可通过"上传头像"功能上传图片文件作为个人头像。支持的图片格式包括 JPG、PNG、GIF 等。

6. 留言板

登录后可在首页的留言板区域发表留言，与其他用户交流。所有用户都可以看到留言内容。

7. 注意事项

  • 请妥善保管您的账号和密码
  • 不要在公共电脑上保存登录状态
  • 使用完毕后请及时退出登录
  • 如遇到问题请联系系统管理员

8. 技术架构

  • 后端框架：Flask（Python）
  • 数据库：SQLite 3
  • 前端：HTML + CSS（Jinja2 模板引擎）
  • 运行端口：5000
